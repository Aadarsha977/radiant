## 🎨 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --color-accent: #8b7cf7;        /* Main accent color */
    --color-bg: #0a0a0f;            /* Background */
    --color-text: #ffffff;          /* Primary text */
}
```

### Fonts
Replace Google Fonts in `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR-FONT&display=swap" rel="stylesheet">
```

### Content
Update business details in:
- `index.html` - All text content
- Schema markup in `<script type="application/ld+json">`
- Meta tags in `<head>`
